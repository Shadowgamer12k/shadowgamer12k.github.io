ABOUT

The Land Claim v1.2 mod is a powerful tool for Minecraft players who want to protect their builds and belongings from griefers, explosions, and unwanted interactions. With this mod, you can claim and secure areas, ensuring that no other players can break or place blocks, open chests, or interact with your structures unless given permission.

-> # Option 1 - Find By Key:

This option allows you to search for a claimed area using a specific key. Players can use this feature to easily locate their claimed regions without manually searching the map.

-> # Option 2 - Areas Limit:

This option is used to display the maximum number of areas a player can claim. It helps manage space and prevents players from monopolizing too many areas on the server.

-> # Option 3 - All Areas:

This option displays all claimed areas on the server, allowing admins or players to view a complete list of claimed regions, whether owned by them or others.

-> # Option 4 - Clear All Areas:

This option allows the removal of all claimed areas at once. It is typically used by administrators to clear reserved spaces and reset claimable areas for all players.

INSTALLATION GUIDE

  1. Download both the Behavior Pack and Resource Pack
  2. Open Minecraft and navigate to Settings > Global Resources
  3. Click on "Import" and select the downloaded .mcpack files
  4. Create a new world or edit an existing one
  5. Under world settings, enable both the Advanced Waypoint Behavior Pack and Resource Pack
  6. Ensure "Require Trusted Packages" is disabled if you encounter any issues
  7. Start your world and use /waypoint help to begin

SUPPORT

Having trouble with the land Claim Add-on? Here are some common troubleshooting steps:

  • Ensure both the Behavior Pack and Resource Pack are activated
  • Check that you're running Minecraft Bedrock 1.20 or higher
  • Verify that experimental features are enabled in your world settings
  • Try reinstalling both packs if commands aren't working
  • For multiplayer issues, make sure all players have the add-on installed
